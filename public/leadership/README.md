# Leadership photos

Drop headshots here using the exact filenames below (JPG or PNG — if you use
PNG, update the `imageUrl` extension in `prisma/seed.ts` to match, then
re-run `npm run db:seed`).

- Recommended: square, at least 400×400px, face centered.
- If a file is missing or fails to load, the site automatically falls back
  to a colored initials avatar — nothing breaks if you leave some blank.
- The same person reuses one photo across both GB terms (e.g. Navin Gupta
  appears in 2022–24 and again as President in 2025–27 — one file covers both).
