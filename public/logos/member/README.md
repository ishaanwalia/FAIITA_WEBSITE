# Member association logos

Logos for verified FAIITA member associations (city/district bodies under a
state chapter). Each file is referenced by `logoUrl` in
`lib/member-associations.ts` — filenames are case-sensitive on Vercel, so the
reference must match exactly. Missing/failed logos fall back to a generic
building icon automatically.
